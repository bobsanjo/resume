118.29 - SDKAGoodService.IsxyzFileByPath Method

This functions checks if the xyz file in the path that is specified is a valid xyz file. If the path that is passed in is a valid path, it creates a xyz file object and performs a series of validations on this xyz file. These validates include:

Validation of page list to ensure that the correct number of pages are seen in this page list file

Sanity check on each page within the page list

Sanity check on the metadata.json files

Sanity check on the xyz log files
