Welcome to the Example

  • These pages demonstrate how to use markdown (.md) with Doxygen to build HTML that is then placed on GitHub. If you don't want to use GitHub and prefer your company server, you can wrap these pages and place them on your sever.
  • Doxygen is a free tool for generating docs from annotated source code. Markdown is a simple, free, text-based format, thus no proprietary authoring tools. GitHub is widely used and very low cost.
  • I almost exclusively author in Textpad. but sometimes I use Atom because it is designed for mardkown. I do find, however, that I always use Textpad to do cleanup when tools like Atom "break" which they always do.
